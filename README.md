# Memory Card Game

A modern, responsive **memory matching game** built using **React + Vite**.  
The goal is simple: flip the cards, match pairs, and complete the game in the fewest moves possible.

Designed with a clean UI, smooth animations, and mobile-friendly responsiveness.

## Tech Stack

- **React**
- **Vite**
- **CSS (Custom, no frameworks)**
- **Vercel** (Deployment)

## Features

- Interactive card flip animations
- Match tracking logic
- Move counter
- Reset game functionality
- Fully responsive (desktop, tablet & mobile)
- Fast build & load times using Vite
